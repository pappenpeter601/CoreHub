Wie brauchen ein eigenes Warenwirtschaftssystem. Zunächt ein eeinfache php basierte WebApplication, perspektivisch soll es aber eine Cloud Anwendung (z.B. AWS) werden.
Die ANforderungen für die Cloud stehen noch nicht fest, außer, dass die Datenmodel später in die Cloud übernommen wird (mit Daten)

Als Webanwendung werden wir eine MarinaDB10 verwenden, die Webanwendung wird php, html,js basiert sein.
Die gesamte Implementierung (Code) ist in Englisch, Die Angezeigte Sparche muss gleich zweisprachig (Deutsch & Englisch) erfolgen. Weitere Sprachen optional hinzufügbar. 

Wir brauchen eine Benutzerverwaltung
- Benutzer haben 1-n Rollen
- Rollen können an funKtionen gelinkt werden. Der Link beinhaltet (create/update/delete)
- Jede Einzelanwendung, in unserer implementation jedes x.php file hat einen Funktionscheck (create/update/delete)
--> Hier bitte state of the Art eine flexible User-Verwaltung erstellen.
